## Getting Started

### Dependencies

- Requires only a C++ compiler and an IDE like VS Code editor.

### Executing program

- Runs the C++ file in terminal
- Go the codes location in Terminal and use the follwing command lines to execute.

```
g++ "filename.cpp" -o "filename"
./filename
```
